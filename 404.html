Informações Gerais

Está aplicação é um sistema de geração de usuários, para testes de sistemas, ou seja, aqui você pode gerar usuários fictícios para usar na sua aplicação.

Com o botão "Gerenciar Atributos", você pode selecionar que atributos quer que esse usuário tenha, assim se adaptando ao sistema onde ele será inserido.

Caso você precise implementar esses usuários em tempo de execução, veja como chamar nossa API no botão:

Chamar Nossa API
